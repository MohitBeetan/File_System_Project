# File_System_Project
A File System Recovery and Optimization Tool that simulates disk management, recovers lost data, and improves file storage efficiency.
This project is a File System Recovery and Optimization Tool designed to simulate how operating systems manage files and storage.

It provides features such as:
- File creation, deletion, and storage management
- Free space management techniques
- Simulation of disk crashes and recovery methods
- Optimization of file read/write operations

The main objective of this project is to demonstrate the working of file systems in operating systems and provide a practical understanding of data recovery and storage optimization techniques.

This project is developed as part of an Operating System academic project.
